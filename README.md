## 📋 项目介绍

密域安全管理系统 (CryptoVault) 是一款致力于提供安全、可靠的加密存储与文件管理的桌面端应用。系统旨在确保用户的核心数据免受未经授权的访问与泄露，为个人与企业提供金融级的本地数据防护。

通过底层的**高性能加密引擎**与**虚拟文件系统 (VFS)**，CryptoVault 能够在用户无感知的情况下，对各种类型的文件（纯文本、代码、Office 文档、PDF、音视频媒体等）进行高强度的加密保护。用户可以像在普通文件管理器中一样操作文件，但所有落地数据均被隔离在安全的加密容器（`.cvlt` 文件）中。

## 🌟 核心特性与技术亮点

### 1. 🚀 高性能加密引擎
- **独立原生核心**：底层加密逻辑与文件外发分享功能采用高性能原生编译核心实现，处理速度极快。
- **国密算法标准**：采用国家商用密码算法 SM4，结合 Copy-on-Write（写时复制）策略，有效防范双重异或攻击，保障“一文一密”（独立文件加密密钥）。

### 2. 🗂️ 隔离的虚拟文件系统 (VFS)
- **沙盒加密容器**：所有用户文件均存储于以 `.cvlt` 为后缀的加密容器中，并辅以自动备份机制，支持多容器无缝切换。
- **容量防绕过机制**：所有文件写入操作在底层汇聚并进行严格的容量合法性拦截。
- **碎片自动压缩**：内置空间清理机制，在退出挂载或手动清理时集中执行压缩，避免后台静默压缩导致的界面阻塞。

### 3. 📝 智能文件预览与协同编辑
- **外部透明编辑**：采用创新的虚拟挂载策略。在调用外部 Word、Excel、PPT 等应用时，系统将文件挂载至安全隔离的虚拟磁盘，同时精准追踪后台进程树，确保文件修改实时安全写回加密容器。
- **内置文本防丢失**：集成高性能文本编辑器进行代码与文本预览，具备静默自动保存与关闭兜底提示，防止意外退出导致的数据丢失。
- **办公套件集成**：支持在挂载盘内创建独立的工作区会话，供各类办公套件进行安全的转换和访问。

### 4. 🛡️ 极高的安全性设计
- **设备指纹与防篡改授权**：采用离线授权激活模式。授权文件采用 RSA 非对称数字签名保护，核心校验逻辑硬编码于安全底层，任何对配置的明文修改都会被直接拒绝。
- **系统休眠与锁屏联动**：系统实时监听操作系统的休眠与锁屏事件，触发时自动卸载加密容器并强制退出登录。
- **内存安全擦除**：退出登录时，系统会将常驻内存的加密密钥进行物理级安全覆写，防止攻击者通过内存 Dump 提取密钥。

### 5. 📦 安全包外发与流转
- **专属封装格式**：支持将容器内文件导出为专用的加密安全包格式（.pkg），在微信、邮件等公共渠道传输时不泄露原文件内容。
- **动态重加密**：导出安全包时，系统会对缺失独立密钥的早期文件自动生成新密钥并进行密文重加密，确保导入方能正确解密。

## ✈️ 主要使用场景

1. **企业核心资产保护**：研发代码、财务报表、战略规划等敏感文档的本地防泄密隔离存储。
2. **断网/离线机密办公**：在无网络环境下，通过机器码指纹绑定实现严格的设备级访问授权。
3. **安全数据流转**：跨部门或对外提供敏感资料时，通过安全包 `.pkg` 格式外发，并限制使用权限与生命周期。
4. **防偷窥与意外泄露**：借助锁屏自动锁定功能，防止员工暂时离开工位时被他人窃取屏幕数据。

## 📒 快速开始

- **系统初始化**：首次启动需提取本机“设备指纹机器码”，联系管理员获取“授权激活码”，设置并确认“系统保护密码”以完成激活。
- **日常登录**：使用设置的系统保护密码登录系统。
- **管理与操作**：在主工作区中，您可进行容器的“新建”、“切换”、“备份”和“恢复”。在文件区可执行导入、新建模板（系统自动写入防损坏极简模板）、外部编辑等操作。
- 详细操作步骤请参考项目随附的《CryptoVault_用户操作手册》。
